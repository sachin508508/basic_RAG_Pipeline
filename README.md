# Basic RAG Pipeline
A Retrieval-Augmented Generation (RAG) pipeline built with LangChain, Chroma DB, and DeepSeek's model.

## Overview

This project demonstrates a Retrieval-Augmented Generation (RAG) pipeline that retrieves relevant information from PDF documents using vector embeddings and generates answers with an LLM.

## Features

- PDF Loading
- Text Chunking
- Embedding Generation
- Vector Search
- Context Retrieval
- LLM Response Generation
